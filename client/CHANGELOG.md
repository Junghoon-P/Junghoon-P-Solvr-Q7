# Client Changelog

### Added

- React 18 + Vite 대시보드 구현
- 4개 차트 컴포넌트
- 타입 안전한 API 호출
- 반응형 UI 디자인
- 실시간 데이터 로딩

### Components

- `Dashboard` - 메인 대시보드
- `ReleaseTrendByMonth` - 월별 트렌드

- `ReleasesByTimeSlot` - 시간대별 분포
